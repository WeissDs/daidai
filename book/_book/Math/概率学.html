1.概率

排列组合公式

排列：从n个元素中取出m个元素，进行排列的方法数 A(n,m)=n!/(n-m)!

function A(n,m){
    let result = 1;
    while(m){
        result*=n;
        n--;
        m--;
    }
    return result;
}

组合：从n个元素中取出m个元素，进行组合的方法数 C(n,m)=n!/(n-m)!m!

function C(n,m){
    let result = 1;
    let f=1;
    while(m){
        result*=n;
        f*=m;
        n--;
        m--;
    }
    result/=f;
    return result;
}

推导：先排列后除以重复情况的排列m!
