3.React教程

React笔记

项目创建

全局安装create-react-app

npm install -g create-react-app

创建test1项目

create-react-app test1

运行项目

npm start

webpack的修改路径

\node_modules\react-scripts\config

修改项目端口

package.json文件

"scripts": {
"start": "set PORT=5000 && react-app-rewired start",

}
